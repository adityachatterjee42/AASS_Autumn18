# Usage
<!-- usage -->
```sh-session
$ npm install -g poparty
$ poparty COMMAND
running command...
$ poparty (-v|--version|version)
poparty/0.0.0 win32-x64 node-v8.12.0
$ poparty --help [COMMAND]
USAGE
  $ poparty COMMAND
...
```
